# HTML-Course

## Основные требования:
1. Макет должен быть сверстан с применением фреймворка [bootstrap4](https://bootstrap-4.ru/)
    * использзование утилитарных классов например, [.border](https://bootstrap-4.ru/docs/4.4/utilities/borders/), [.text-primary](https://bootstrap-4.ru/docs/4.4/utilities/colors/), [.d-flex](https://bootstrap-4.ru/docs/4.4/utilities/flex/)
    * использзование компонентов например, [.btn](https://bootstrap-4.ru/docs/4.4/components/buttons/), [.card](https://bootstrap-4.ru/docs/4.4/components/card/), [.jumbotron](https://bootstrap-4.ru/docs/4.4/components/jumbotron/)
1. Кроссбраузерность
1. Семантическая верстка
1. БЭМ-методология
1. Всегда описывайте цвет фона для body даже если он белый например, bg-white, bg-primary
1. Названия классов должны по смыслу соответствовать применению например, header, menu, footer, news
1. Делать коммиты после завершения кождого блока
1. Просьба разделять основные html блоки комментариями
1. CSS файл должен быть разбит с помощью строк с комментариями на блоки по функциональному назначению, например:
```
/* 1. Сброс CSS */
/* 2. Типовые элементы */
/*  2.1. Залоговки */
/*  2.2. Ссылки */
/*  2.3. Элементы форм*/
/* 3. HEADER (Шапка сайта) */
/* 4. FOOTER (Подвал ) */
/* 5. SIDEBAR (Справа) */
```
